# 1.1.2、RRAS

:::danger[说明]
1. 操作不当可能会导致网络无法访问，请谨慎操作，最好先创建还原点
2. 有局限性，创建虚拟网卡后，需要重新`“配置并启用路由和远程访问”`
:::

1. 在windows server中，你可以选择使用RRAS(Routing and Remote Access Service) 来启用NAT，从而实现点对网
